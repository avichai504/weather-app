## How to Run

1. Clone the project:

   ```bash
   git clone https://github.com/avichai504/weather-app
   ```
2. Install dependencies:

   ```bash
   npm install
   ```
3. Start the development server:

   ```bash
   ng serve
   ```
4. Open your browser at:

   ```
   http://localhost:4200
   ```
